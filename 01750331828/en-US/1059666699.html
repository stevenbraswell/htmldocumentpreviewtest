Removal

Right side function keys are defined as you face the fascia.

  1. Unlock and completely extend the fascia on its slide assemblies.
  2. Using T/SOP, perform the module shut-down procedure (refer to Modules - Shutting Down and Starting Up).
  3. Disconnect the fan cable.
  4. Remove the two screws and nuts that secure the fan (Fan Bracket (on fascia)).
  5. Remove the fan.
  6. Disconnect the ribbon cable for the right side function key assembly from the EPP soft key cable (Function Keys (right side)).
  7. Remove the three screws from the function key assembly.
  8. Remove the right side function key assembly from the touch screen mounting bracket out the front of the fascia.
NOTICE

Retain the function key shim for reuse with the replacement function key assembly.
